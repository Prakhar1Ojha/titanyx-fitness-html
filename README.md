# 🏋️ Titanyx Fitness Gym

A complete, multi-section HTML5 website for a fictional premium fitness center — built as a pure HTML + CSS framework + JavaScriptpractice project.

---

## 📖 Overview

Titanyx Fitness Gym is a static single-page website that simulates the full online presence of a commercial gym. It covers everything a real gym site would need: membership info, trainers, schedules, nutrition plans, forms, and contact details — all built using semantic HTML5,CSS framework and JavaScript.

This project was created to practice structuring a large, realistic website using proper HTML5 elements instead of relying on generic `<div>`s.

---

## ✨ Features

- Responsive-ready meta setup (viewport, charset, SEO meta tags)
- Hero section with call-to-action buttons
- About, Why Choose Us, and Facilities sections
- Gym statistics & department overview tables
- Membership plans with pricing & benefits comparison tables
- Trainer profiles + trainer directory table
- Workout programs, training levels (`<dl>`), and equipment inventory
- Nutrition & supplement guides with meal plan tables
- BMI calculator form (UI only, JS not yet connected)
- Weekly class schedule & trainer availability tables
- Gallery using `<figure>`/`<figcaption>`
- Member transformations, testimonials (`<blockquote>`), success stories
- FAQ section using `<details>`/`<summary>` (no JS needed)
- Contact form + newsletter signup form
- Branches table, business `<address>`, and map placeholder
- Social media links and quick navigation
- Footer with copyright, contact info, and back-to-top link

## 🚀 How to Run

1. Clone or download this repository.
2. Open the folder in VS Code (or any editor).
3. Open `TFGym.html` directly in your browser, or use the **Live Server** extension for auto-reload.

No build steps, no dependencies .
